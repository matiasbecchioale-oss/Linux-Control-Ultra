## 📖 Acerca del proyecto
**Linux-Control-Ultra** es un panel de control moderno diseñado para simplificar la administración de sistemas Linux. A diferencia de las herramientas tradicionales, combinamos una interfaz de usuario (UI) de alta fidelidad con potencia de backend para monitorear recursos en tiempo real, gestionar servicios y optimizar el sistema con un solo clic.

> "La administración de sistemas no tiene por qué ser aburrida. Creamos una herramienta que se siente como software de escritorio profesional."

## ✨ Características principales
* 📊 **Telemetría en tiempo real:** Gráficas dinámicas de CPU, RAM, Temperatura y Red.
* ⚙️ **Gestor de Systemd:** Control total de servicios con edición directa.
* 🛡️ **Seguridad proactiva:** Auditoría y firewall en una sola pantalla.
* 🚀 **UI/UX Moderna:** Diseño inspirado en las mejores interfaces del mercado (tokens de diseño optimizados).
* 🔧 **Modo simulación:** Prueba cambios de configuración antes de aplicarlos al sistema real.

## 🛠️ Stack Tecnológico
* **Backend:** Python (Core logic & System API interaction)
* **Frontend:** Web-based UI (HTML5, CSS3, JS)
* **Integraciones:** Systemd, Netstat, Kernel Sysctl

## 🚀 Instalación rápida
1. Clona el repositorio:
   ```bash
   git clone [https://github.com/matiasbecchioale-oss/Linux-Control-Ultra.git](https://github.com/matiasbecchioale-oss/Linux-Control-Ultra.git)
   ```
2. Ejecuta el script principal:
   ```bash
   python3 linux_control.py
   ```
